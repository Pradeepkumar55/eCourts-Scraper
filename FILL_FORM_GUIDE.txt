║                    STEP-BY-STEP INSTRUCTIONS                           ║
╚════════════════════════════════════════════════════════════════════════╝

  STEP 1: Start the Web Application
  ──────────────────────────────────
  • Double-click: START_WEB.bat
  • Wait for "Running on http://localhost:5000"
  • Open browser and go to: http://localhost:5000

  STEP 2: Fill the Search Form
  ─────────────────────────────
  • Copy CNR from above (e.g., ABCD1234567890123)
  • Paste into "CNR Number" field
  • Enter Court ID: 101
  • Select "Today" from "When to Check" dropdown
  • (Optional) Enter State and District

  STEP 3: Submit and View Results
  ────────────────────────────────
  • Click "Search Case Listing" button
  • Wait for results page to load
  • See case details with serial number
  • View party names, advocate, and purpose


╔════════════════════════════════════════════════════════════════════════╗
║                    TROUBLESHOOTING                                     ║
╚════════════════════════════════════════════════════════════════════════╝

  Problem: "Court ID is required" error
  Solution: Make sure you filled in the Court ID field (101 or 102)

  Problem: "Case not found" message
  Solution: Check that you copied the CNR correctly (no extra spaces)

  Problem: Web page won't load
  Solution: Make sure START_WEB.bat is still running

  Problem: Form validation error
  Solution: Either fill CNR OR fill Case Type/Number/Year (not both)
