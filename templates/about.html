  • 🔌 API Access: JSON API endpoint for programmatic access

📝 Example API Request

curl -X POST http://localhost:5000/api/search \
  -H "Content-Type: application/json" \
  -d '{
    "cnr": "ABCD1234567890123",
    "when": "today",
    "court_id": "101"
  }'
